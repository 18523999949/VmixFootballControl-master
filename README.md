# 足球比赛字幕控制系统

一个专业的足球比赛字幕控制系统，与 vMix 软件无缝集成，用于实时管理比赛比分、换人、红黄牌、进球等字幕显示。

## 🎯 主要功能

- **比分管理** - 实时调整主队/客队比分，切换比赛时段
- **换人记录** - 快速记录换人信息并生成字幕
- **红黄牌管理** - 记录红牌/黄牌并自动生成字幕
- **进球记录** - 记录进球球员并自动更新比分
- **球队设置** - 自定义球队名称、颜色和球员名单
- **vMix 集成** - 自动连接 vMix，实时控制字幕显示

## 🚀 快速开始

### 使用预编译版本（推荐）

1. 下载最新版本的 `足球比赛字幕控制系统V*.exe`
2. 确保以下文件与 exe 同目录：
   - `config.json`（配置文件）
   - `home.txt`（主队名单，格式：`号码,姓名`）
   - `away.txt`（客队名单，格式：`号码,姓名`）
3. 双击运行 exe 文件

### Python 源码运行

```bash
python a0.95.py
```

## ⚙️ 基本配置

### 1. 连接 vMix

1. 启动 vMix 软件
2. 确保 vMix API 已启用（Settings → API & Web）
3. 在程序中点击 **"[连接]"** → **"连接 vMix"**
4. 默认连接地址：`127.0.0.1:8099`

### 2. 设置球队

1. 点击 **"[设置]"** 按钮
2. 设置主队/客队名称和颜色
3. 准备球队名单文件（`home.txt` 和 `away.txt`）

### 3. 使用功能

- **比分**：左上角记分板，点击 `+`/`-` 调整比分
- **换人**：点击 **"[换人]"**，输入 `换下号码 换上号码`，点击 **"上字幕"**
- **红黄牌**：点击 **"[牌]"**，输入球员号码，选择红牌/黄牌，点击 **"上字幕"**
- **进球**：点击 **"[进球]"**，输入球员号码，点击 **"上字幕"**（比分会自动更新）

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `config.json` | vMix 连接和字幕配置 |
| `home.txt` | 主队球员名单（格式：`号码,姓名`） |
| `away.txt` | 客队球员名单（格式：`号码,姓名`） |
| `scoreboard.csv` | 比分数据（自动生成） |
| `goal.csv` | 进球记录（自动生成） |
| `red_card.csv` | 红牌记录（自动生成） |
| `yellow_card.csv` | 黄牌记录（自动生成） |
| `substitutions.csv` | 换人记录（自动生成） |

## ❓ 常见问题

**Q: 无法连接到 vMix？**  
A: 确保 vMix 已启动，API 已启用，IP 和端口配置正确。

**Q: 字幕没有显示？**  
A: 检查 vMix 中的输入层编号和字幕模板配置。

**Q: CSV 文件数据丢失？**  
A: 程序启动时会自动清空事件记录文件，这是正常行为。
